# ChronoLuxe - Premium Luxury Watches

A global destination for premium luxury watches from the world's most trusted brands. Authentic timepieces with worldwide shipping & warranty.

## Features

- **Premium Watch Collection** - Curated luxury timepieces from Rolex, Omega, TAG Heuer, and more
- **HD Quality Imagery** - High-resolution product images (1600px+)
- **Responsive Design** - Beautiful UI that works on all devices
- **Fast Performance** - Built with TanStack Start and Vite for optimal speed
- **Customer Care Pages** - Comprehensive shipping, returns, warranty, FAQ, and support sections

## Tech Stack

- **Framework**: TanStack Start (React)
- **Styling**: TailwindCSS
- **UI Components**: Radix UI + shadcn/ui
- **Icons**: Lucide React
- **Build Tool**: Vite

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
